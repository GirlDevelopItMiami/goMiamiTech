Miami Tech Events

Inspiring the next generation of developers and innovators in Miami.

Code in Costume: Pirate Blunder

3/7/15 - 3/8/15: You heard that right. Now keep in mind that the costume is totally optional but the venue and festivities will keep with the theme as well. When else are you gonna get dressed up to write code? Ideas could include a pirate themed mobile or Facebook game. The participants of the event will get food, prizes, sponsor goodies, lots of fun and great networking opportunities!

South Florida Code Camp

2/7/15: Code Camp is a FREE one day GEEK FEST. This is the 11th year for South Florida Code Camp. The event will have speakers from the local community and beyond presenting some of the most requested topics like Windows 8, Windows Phone 8, ASP.NET, Javascript, JQuery, Visual Studio 2013, MVC Framework, Sharepoint 2013, SQL Server 2014, Xamarin, cross platform development and Azure. Code Camp is community driven by a group of dedicated volunteers and speakers. Fifteen tracks, over 80 sessions, 70 speakers and breakfast and lunch is included!

SunshinePHP Conference

2/5/15 - 2/7/15: This event is hosted by the South Florida PHP community (SoFloPHP) and includes the best speakers, awesome talk topics, latest technologies, and up to date news in PHP. And don't forget our Hack-a-thon and Uncon'ference, as well as a great hallway track! The conference has something for every level of PHP developer.

Sime MIA

12/1/2014 - 12/3/2014: Sime MIA brings together keynote speakers, global thought-leaders from technology, media and the arts to Miami to network and exchange their vision for the future. The Summit is sprinkled with inspiring performances and surprises that will delight 600+ attendees from around the world.

ITPalooza

12/04/2014: ITPalooza is an annual gathering of South Florida's non-profit Technology User Groups hosted at at Nova Southeastern University in Davie. In addition, it includes a Hiring Fair, Technical Sessions and Vendor exhibits, and a dedicated management track for CIOs.
